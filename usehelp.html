MRP虚拟机 Version 2500
Design by 王耀国春飞wygcf
「软件介绍」
keyboard_arrow_down

MRP虚拟机 ( MRPEMU / MRPOID ) 是安卓平台上的老牌经典模拟器，高版本采用 Material Design 规范设计，拥有像安卓老牌管理器 Root Explorer 的自研算法文件浏览器功能，同时内置极速浏览插件与mrp标签编辑器，是一款功能强大的实用管理软件，拥有独具特色的 mrp软件游戏 运行功能，让你可以在安卓设备上高效地进行各种MTK手机上的 mrp文件 运行操作以及编辑修改。

「官方申明」
keyboard_arrow_down

1. MRP虚拟机 主要为热爱 mrp软件游戏 的用户们提供便利，以及作为日常 mrp文件管理器 使用。在使用过程中，如果由于用户的操作导致系统损坏、文件丢失等问题造成的损失，一律由用户自行承担责任。

2. MRP虚拟机 的 mrp修改功能 主要为了方便用户在手机上学习了解 mrp软件游戏 、mrp DIY 以及 正/逆向工程师分析 mrp文件，请勿将此功能用于任何非法用途。

3. 尊重他人版权，如果您使用 MRP虚拟机 修改了他人的 mrp应用，在公开发布前，请务必先取得原开发者的同意，如果因此产生任何纠纷，我们不承担任何责任。

「功能说明」
keyboard_arrow_down

1. 对 mrp文件 标签编辑、规范命名、创建快捷方式、按照尺寸运行、删除。

2. 像 RX管理器 那样管理运行 mrp 格式文件，支持 [ext] 和 [mr] 两种函数方式启动。同时支持多个 mrp应用 同时后台运行(支持后台运行暂停)。

3. 自带强大的 mrp解释器，支持触屏、按键同时使用。同时支持 编辑按键布局、编辑按键颜色，也能模拟不同设备机器型号进行优化，支持识别错误 mrp应用文件，支持界面字体更改。

4. 拥有像 Root Explorer 那样的文件浏览，同时具备 Open Graphics Library 设置接口，支持 hqx 滤镜，支持定义机器内存、拓展模拟运行内存，支持屏幕 缩放拉伸、截图，支持应用运行调速。

5. 带有像系统那样的 屏幕常亮、开放 Frames Per Second 接口、按键振动、全屏显示、消除混叠、屏幕常亮、显示系统标题栏窗、音量按键模拟映射，这些人性化的功能。

6. 支持设置默认主页( 冒泡商店 和 本机应用 的更改 )，自带的极速浏览器( 支持 HTML5,CSS3,JS 语言 )，内置有意思的 打赏支持、退出软件 功能。
